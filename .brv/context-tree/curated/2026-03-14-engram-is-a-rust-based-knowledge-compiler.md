## Raw Concept

Engram is a Rust-based knowledge compiler and query engine for AI agent memory. It transforms Markdown files with YAML frontmatter into a BM25-searchable Tantivy index with multi-tier caching, causal graph traversal, temporal queries, and governance controls.
