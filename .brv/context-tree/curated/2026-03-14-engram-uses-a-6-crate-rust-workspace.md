## Raw Concept

Engram uses a 6-crate Rust workspace: engram-core (schema/parsing), engram-bulwark (policy/audit), engram-compiler (indexing), engram-query (search/caching), engram-openclaw (plugin interface), and engram-cli (binary entry point).
